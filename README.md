# Speech Recognition using Python

A Speech Recognition project developed using Python and the SpeechRecognition library. The project converts speech into text from audio files as well as real-time microphone input. It also demonstrates noise reduction techniques and a voice-controlled YouTube search application.

## Features

* Convert speech from audio files to text
* Convert speech from specific audio segments using duration and offset
* Real-time speech recognition using a microphone
* Noise reduction using ambient noise adjustment
* Voice-controlled YouTube search
* Google Speech Recognition API integration

## Technologies Used

* Python
* SpeechRecognition
* PyAudio
* BeautifulSoup4
* VLC Media Player

## Installation

1. Clone the repository

git clone https://github.com/Rohitkumar78/speech-recognition.git

2. Move into the project directory

cd speech-recognition

3. Install dependencies

pip install -r requirements.txt

## Usage

Run the notebook or Python script and follow the instructions to:

* Convert speech from an audio file
* Capture speech from a microphone
* Test noise reduction
* Search YouTube videos using voice commands
